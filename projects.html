Project 4: Auction System

This project is a distributed auction system which allows sellers to create a username and password, and list items to auction. It also allows buyers to create credentials as well and login and bid on different items listed. It uses Java RMI to create and use a remote registry, signal files for communication and to indicate different commands needed such as adding or terminating replicas. It uses active replication by implementing JGroups for fault tolerance by having mulitiple replicas in case of replica/server failure. This project was developed using Java, and JGroups.
